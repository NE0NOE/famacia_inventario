# Farmacia Dulce Esperanza - Sistema de Gestión Integral

Bienvenido al repositorio oficial del proyecto **Farmacia Dulce Esperanza**. Este sistema es una solución completa de software diseñada para modernizar y optimizar la gestión operativa de una farmacia.

##  Características Principales

El sistema está compuesto por tres módulos interconectados:

1.  **Panel Administrativo Web (Farma Web):**
    *   Gestiona el inventario de medicamentos y productos.
    *   Administra proveedores y clientes.
    *   Genera reportes de ventas y analiza el rendimiento del negocio.
    *   Punto de Venta (POS) rápido y eficiente.

2.  **Aplicación Móvil (Farma Mobile):**
    *   Herramienta de apoyo para el personal de piso.
    *   Consulta rápida de precios y stock.
    *   Registro de nuevos clientes y consulta de puntos de fidelidad.

3.  **Backend API (Farma Backend):**
    *   Núcleo del sistema construido con Node.js y Express.
    *   Base de datos PostgreSQL robusta y segura.
    *   Autenticación y manejo de sesiones.

##  Tecnologías Utilizadas

*   **Frontend:** React, Vite, Tailwind CSS.
*   **Backend:** Node.js, Express, PostgreSQL.
*   **Móvil:** Capacitor (Android).
*   **Escritorio:** Electron.
